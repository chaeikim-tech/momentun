# Momentun

**HTML & CSS & JavaScript** 로 만든 Momentum Clone Web 입니다.<br>
Momuntum 의 기본 기능인 ToDo List와 현재시각, 사용자 이름 설정 및 날씨 기능을 구현하고,
사이드 메뉴가 Click 이벤트로 인해 슬라이드 하는 기능을 JavaScript를 이용하여 더해서 완성했습니다.
